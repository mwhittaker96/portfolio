Vicarious Visions | Blizzard Entertainment 路 2021

What a game to start my career with! Not just any remaster, the UI team built a 3d engine on top of the legacy game in C/C++, preserving the original game while making many new improvements like accessibility settings and gamepad support and allowing seamless switching between the 2d and 3d world.

Key Contributions

  • Contributed core elements to modern UI system built on top of heavily memory constrained legacy game.
  • Built new features like comparison tooltips, with dynamic layout system that adapted gracefully to user accessibiltiy settings and screen sizes ranging from TVs to the Nintendo Switch.
  • Worked with social features like chat, friends list, and multiplayer trading.
  • Wrote recycler view for infinite (or nearly infinite) scroll UI like chat and friends list.
